# BusinessLink — Art, Mode & Culture

Site vitrine de **BusinessLink**, la branche services d'[Art, Mode & Culture](mailto:contact@artmodeetculture.com) basée à **Lille**.

## Présentation

BusinessLink propose un accompagnement complet aux professionnels et associations :

- **Communication** — création de supports print et visuels
- **Web** — conception et développement de sites internet
- **Administratif** — accompagnement dans les démarches administratives
- **Traduction** — services de traduction
- **Développement commercial** — aide à la croissance de votre activité

Un seul interlocuteur pour faire avancer votre activité.

## Structure du projet

```
businesslink/
├── index.html          # Page unique (one-page)
├── css/
│   └── style.css       # Styles — palette noir & or (charte AM&C)
├── js/
│   └── main.js         # Navigation mobile, animations, formulaire
└── assets/
    ├── logo.svg        # Logo BusinessLink
    └── favicon.svg     # Favicon
```

## Stack technique

- **HTML5** sémantique + schema.org (référencement local)
- **CSS3** — variables CSS, design fluide (clamp), dark theme noir & or
- **JavaScript** vanilla — aucune dépendance externe
- Polices Google Fonts : [Jost](https://fonts.google.com/specimen/Jost) + [Cormorant Garamond](https://fonts.google.com/specimen/Cormorant+Garamond)

## Démarrage rapide

Ce projet est un site statique, aucune installation n'est nécessaire.

```bash
# Ouvrir directement dans le navigateur
open index.html

# Ou avec un serveur local (recommandé)
npx serve .
# → http://localhost:3000
```

## Déploiement

Le site peut être hébergé sur n'importe quelle plateforme de fichiers statiques :

- **GitHub Pages** — activer dans *Settings → Pages → Deploy from branch*
- **Netlify / Vercel** — glisser-déposer le dossier ou connecter le dépôt
- **Hébergement FTP classique** — uploader les fichiers tels quels
